# Personal Website (Static)

This is a simple, fast personal website generated from your LinkedIn profile PDF.

## Files
- `index.html` – main page
- `styles.css` – styling (dark/light mode via CSS variables)
- `script.js` – mobile nav, theme toggle, and a demo contact form
- `profile.png` – your profile photo cropped from the PDF

## Run locally
Open `index.html` directly in a browser, or run a local server:

### Python
```bash
python -m http.server 8000
```

Then open http://localhost:8000

## Deploy options
- GitHub Pages (static)
- Netlify / Vercel (static)
- Any web host that can serve static files

## Customize
- Update experience bullets in `index.html` for more project impact.
- Add a `projects` section if you want (I can generate it from your resume/project list).
